# 🎉 Sistema de Cadastro de Usuários

Projeto desenvolvido com o objetivo de aplicar conceitos de desenvolvimento web, versionamento de código e boas práticas utilizando GitHub.

---

## 🚀 Tecnologias utilizadas

- Python
- Flask
- HTML5
- Bootstrap 5
- Git e GitHub

---

## 💻 Sobre o projeto

O sistema permite realizar o cadastro de usuários de forma simples, armazenando informações como nome, idade e e-mail, além de exibir uma lista com os usuários cadastrados.

Este projeto foi desenvolvido como atividade acadêmica, com foco na prática de:

- Controle de versão
- Criação de branches
- Merge de alterações
- Organização de código

---

## ⚙️ Funcionalidades

✔️ Cadastro de usuários  
✔️ Listagem de usuários cadastrados  
✔️ Interface responsiva com Bootstrap  
✔️ Versionamento com GitHub

## ▶️ Como executar o projeto

<img align="right" width="450" src="https://raw.githubusercontent.com/SimoneSoaress/SimoneSoaress/main/SimoneCorpoInteiro.png" style="border-radius: 50%;">

1. Clone o repositório: git clone https://github.com/SimoneSoaress/sistema-cadastro.git


2. Acesse a pasta do projeto: cd sistema-cadastro


3. Instale as dependências: pip install flask


4. Execute a aplicação: python app.py


5. Acesse no navegador: http://127.0.0.1:5000



### 🔄 Versionamento

Durante o desenvolvimento foram aplicados conceitos de versionamento, incluindo:

- Commit inicial do projeto  
- Criação de branch para melhorias  
- Alterações no front-end  
- Merge com a branch principal  
- Histórico de commits para auditoria
